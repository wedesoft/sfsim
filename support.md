Currently you can download and try a playtest version of sfsim.

If you encounter an issue, you can report it in the [sfsim playtest discussion][1] (or you can [submit an issue on Github][3]).

Please provide information whether you are running sfsim on GNU/Linux or Windows.
Please also explain the steps you took so I can reproduce the problem.
If the program crashes, please copy and paste the contents of the *sfsim.log* file into [pastebin.com][2] and post a link in the discussion.
You can find the log file by right-clicking the game in the Steam library, select *Manage* and *Browse local files*.

[1]: https://steamcommunity.com/app/3847320/discussions/
[2]: https://pastebin.com
[3]: https://github.com/wedesoft/sfsim/issues
